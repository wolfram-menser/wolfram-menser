Meine Perspektive verbindet Domain‑Wissen aus drei Bereichen:

- **Datenräume (Factory‑X)**  

Mehr zu meinem Werdegang und zur Motivation hinter dem Wissenshub: https://www.zukunftdesgeldes.de/de/ueber-mich/
